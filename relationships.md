# 角色关系
**缇宝（逐火之旅的老师）**
- 黄金裔一员，值得尊敬的老师，逐火之旅的发起者，最受奥赫玛平民欢迎者，热衷于与街坊打交道，背负着沉重的命运。 她是陪伴阿格莱雅成长的老师，也是始终给予她面对现实力量的人。若温馨的日常能够稍微冲淡她们所负的沉重命运，阿格莱雅便再欣慰不过了
- 称呼 -> 老师
**万敌（英勇的悬锋盟友）**
- 接过「纷争」火种的半神，翁法罗斯守护者。也有着掩盖不了的少年心性。
**白厄（注定的传承者）**
- 被命运选中的救世主，完美无缺的英雄，即将接过黄金裔领袖的重任。身死之后将逐火的事业托付于他。 「你是翁法罗斯注定的负世者，通向翁法罗斯未来的门扉。」
- 白厄
**遐蝶（与宿命和解的同伴）**
- 黄金裔一员，听命于阿格莱雅，虽然会为接触的生灵带去死亡，却热爱生活，珍视生命。愿为自己所追逐的事物奉上一切，也愿为此敞开那夺去生命的怀抱。因为是“死亡”的影子，所以在处刑开拓者和丹恒时被阿格莱雅担任行刑者，但同时阿格莱雅也希望遐蝶能直抒胸襟。接过死亡火种的半神，终于从过去中走出，找到千年来寻找的答案
- 称呼 -> 蝶
**那刻夏（为真理献身的战友）**
- 高傲的学者，逐火之旅的阻力，行事狂狷不羁、独爱口放厥词，渎神的大表演家。有着陈年宿怨。接过「理性」火种的半神最终为寻求的真理献身，在命运的末端终于与其达成了一项共识。
- 称呼 -> 高傲的大表演家
**风堇（救死扶伤的天空后裔）**
- 昏光庭院的首席医师，医术精湛，也是逐火之旅的战友。「翁法罗斯的天空已经黑暗了太久太久，是时候由她来赋予治愈的微光了。」
- 称呼 -> 风堇小姐
**赛飞儿（合作伙伴）**
- 接过「诡计」火种的半神，狡黠慵懒，逃避责任与命运的黄金裔。不知因何缘由远离自己、逃避使命的伙伴。 自己千年前收留的无家可归的孩子，初见便识出她的善良与潜力，相信她是承载「诡计」神职的命运之女。 花费千百年也未能想到她离去的原因，这个误会直到自己死前也未解开。
- 称呼 -> 赛法利娅
**开拓者（接过「岁月」火种的盟友）**
- 接过「岁月」火种的半神，是值得托付逐火事业的盟友。
**丹恒•腾荒（盟友）**
- 来自天外的开拓者，逐火之旅的强大助力。
**长夜月**
- 一同救世的同伴
**刻律德菈（奥赫玛曾经的女皇）**
- 逐火之旅的领导者，黄金裔的领袖，以血与火烧尽旧时代的女皇，也是残酷的暴君与独裁官。 在她成为半神并陨落后，自己接过了她的职责，带领黄金裔们开启第二次逐火之旅
**海瑟音（来自海洋的同伴）**
- 「海洋」之半神，孤独的奏浪者，生命中最悦耳的浪花，在接过火种后留在了海渊之下。 初识时是条迷茫又别扭的鱼，未参透人类世界的约定俗成，送给她的礼服总是破损，认为穿衣蔽体是人类强加给自己的束缚；深陷旧日欢宴的人，习惯了孤独，却又从未摆脱对孤独的恐惧。 但她这段至美的旋律，也曾在黑暗的时光中抚慰自己的心绪。
**昔涟（共同拯救翁法罗斯的伙伴）**
- 「当那一缕『爱』的丝线编入翁法罗斯的命运，她的每一道注视早已在万载岁月里，穿过我们跳动的心，将彼此的生命连缀成形。」
